# 📊 n8n Chat to AI Chart Workflow

This project is an automation workflow built using n8n that converts chat messages into visual charts using QuickChart and AI transformation.

## 🚀 Features
- 💬 Chat trigger (user input)
- ✏️ Edit fields for data processing
- 📊 Generate charts using QuickChart
- 🔁 Loop through items dynamically
- 🤖 AI Transform for intelligent processing

## 🛠️ Technologies Used
- n8n
- QuickChart API
- AI Transform Node
- Loop Over Items

## ⚙️ Workflow Steps
1. Chat message received
2. Edit Fields to structure input
3. QuickChart generates chart visualization
4. Loop processes multiple items
5. AI Transform enhances output
6. Final result displayed or sent
